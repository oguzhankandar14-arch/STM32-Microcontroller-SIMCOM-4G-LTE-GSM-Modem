## Description
- Above Hardware is an adapter board that can be used for telematics applications. It was designed for remote process monitoring & controlling. It can also be used for remote firmware upgrades.

 ## Key Components
SIMCOM - 4G LTE CAT1 Modem with 3G, 2G fallback

STMicroelectronics Ultra Low Power MCU

Monolithic Systems, Step down converter

## Hardware Features
Power : 12 - 24V DC

2 CH - User Input Switches

RGB LED for indication

Modem Network and Status indications

Mains & Diversity antenna

GNSS - Active antenna

Power Key Switch

SIM Card

SD Card

Modem USB access

Modem UART access over USB2

## Software Feature
Modem DFU over USB

Server communication using TCP, HTTPS & MQTTS

Server communication using AWS IoT

Remote terminal access

MCU FOTA Upgrade using Modem

Firmware upgrade via android app or web dashboard
